Requirimientos>>>>

Servidor

X	Se ejecuta en una maquina virtual
X	Sistema Operativo basado en Linux
X	Recibe archivos de manera secuencial
X	Retorna la cantidad de consonantes
	Retorna las llamadas al sistema (Duracion)
X	Devuelve al cliente 
	Archivo de Configuracion
	Archivo de eventos (Log)
	El servidor inicia cuando se inicia la maquina
X	Implementa las funciones de start, stop, restart y status

Cliente
	
X	El cliente se ejecuta en un contenedor
X	Envia el documento
X	El usuario dijita end para parar el servicio
X	Recibe la respuesta del servidor y al muestra
X	Se configura la ruta del documento, el puerto y la IP
